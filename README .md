# Müşteri Kayıp Analizi

Bu çalışma, bir telekomünikasyon veri seti üzerinden hangi müşterilerin hizmetten ayrılacağını (churn) tahmin etmek için hazırlanmıştır.

## 📋 Proje Özeti
Proje, ham verilerin temizlenmesi, görselleştirilmesi ve **Lojistik Regresyon** algoritması kullanılarak bir tahmin modeli oluşturulması adımlarını kapsar.

## ⚙️ Kullanılan Araçlar
- Python (Pandas, NumPy)
- Matplotlib & Seaborn (Görselleştirme)
- Scikit-Learn (Makine Öğrenmesi)

## 🏃 Çalıştırma
1. Gerekli kütüphaneleri kurun: `pip install pandas numpy matplotlib seaborn scikit-learn`
2. `notebook.ipynb` dosyasını Jupyter veya Google Colab üzerinden çalıştırın.

## 📈 Sonuç
Veri setindeki dengesizlikler (class imbalance) giderilerek yaklaşık **%79** doğruluk oranıyla müşteri kaybı tahmini yapılmıştır.
